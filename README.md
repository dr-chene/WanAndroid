### WanAndroid App

**简介：** 运用WanAndroid开放api结合jetpack打造的一款Android技术文章分享社区app，帮助您系统方便的学习Android技术

本app采用kotlin+mvvm编写，使用了模块化编程、retrofit+flow、jetpack、material design组件等技术
